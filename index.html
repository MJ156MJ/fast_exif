Fast EXIF

Простой и удобный инструмент для очистки метаданных ваших изображений

Что

EXIF-метаданные — это скрытая информация, которую добавляют камеры и программы при сохранении изображений. В неё входят: Геолокация (координаты съёмки). Параметры камеры (модель, диафрагма, выдержка). Данные о программе обработки и многое другое

Как выглядит

Что делает

FastExif Может удалять:

  • Только геотеги.
  • Только информацию о камере
  • Только параметры съёмки
  • Только программу обработки
  • Или всё сразу (полная очистка)
Плддерживает форматы: JPG, PNG, WEBP, TIFF, BMP

Пакетная обработка
  • Одновременная работа с десятками файлов
  • Прогресс-бар для отслеживания статуса обработки
  • Безопасное сохранение — оригинальные файлы остаются неизменными
  • Результат сохраняется в новые файлы с суффиксом _clean (например: photo_clean.jpg)
